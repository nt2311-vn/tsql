# TSQL 0.1.0 Spec

## Goal

Ship the first usable open-source MVP of TSQL: a Rust terminal database client that can validate TOML config, execute SQL scripts against SQLite/Postgres, and provide a minimal Ratatui query editor.

## In Scope

- CLI:
  - `tsql config check --config <path>`
  - `tsql exec --url <url> --file <path>`
  - `tsql exec --config <path> --connection <name> --file <path>`
  - `tsql tui --config <path> --connection <name>`
- Config:
  - TOML named connections.
  - `driver = "sqlite" | "postgres"`.
  - `url = "${ENV_VAR}"` expansion.
- SQL:
  - Preserve pasted multi-line text.
  - Split common SQL scripts by semicolon outside strings/comments/Postgres dollar quotes.
- Database:
  - SQLite execution.
  - Postgres execution.
  - Result model with columns, rows, and rows affected.
- TUI:
  - Catppuccin Mocha default theme.
  - Editor panel.
  - Result panel.
  - Status panel.
  - `Ctrl+R` execute.
  - `Esc`/`Ctrl+C` quit.
- CI:
  - fmt, clippy, tests, audit.
  - SQLite test.
  - Postgres integration test with CI service.
  - existing security workflows.

## Out of Scope

- Full SQL formatter.
- Full SQL lint engine.
- Driver-aware autocomplete.
- MySQL, MariaDB, Oracle, MSSQL.
- Advanced TUI table navigation.
- Connection secret manager integration.

## Release Criteria

- README includes usage, config, roadmap, and release flow.
- CHANGELOG tracks 0.1.0 work.
- justfile includes local dev, test, integration, smoke, and release-check tasks.
- `knowledge.aaak`, `graph.md`, and Vault docs are current.
- All mandatory checks pass.
- Trivy and cargo-audit have no HIGH/CRITICAL dependency findings.
- PR reviewed and merged into `main`.
- Tag `v0.1.0` created from `main`.
- Manual release workflow triggered from protected environment.
